PyConJP 2018 聴講メモ: Jupyterで広がるPythonの可能性
python

jupyter notebook 便利

  • OS のコマンドが、 python のオブジェクトとして保存される
  • vmstat の結果を3行で加工簡単
  • shift tab
  • ワイルドカードでオブジェクトを検索
  • アンダースコアに実行結果を保存できる
    • 確認しながら試しながら書ける
  • ! でコマンド実行できる
  • マジックコマンド
    • % OS 依存排除できる
    • コードの実行時間書ける
      • %timeit
    • 標準出力も拾える
