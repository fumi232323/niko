コマンド
--------

* Automatic rebuilds

  .. code-block:: bash

    $ nikola auto --browser


* Deploying to GitHub

  .. code-block:: bash

    $ nikola github_deploy

* Delete the now orphaned files

  .. code-block:: bash

    $ nikola check --clean-files
