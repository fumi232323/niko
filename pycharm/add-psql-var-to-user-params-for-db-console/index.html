PyCharm の Database Console で PostgreSQL のプレースホルダー (インタポレーション) を SQL パラメーターとして使えるようにする
created: 2018-09-26 Wed updated: 2018-09-26 Wed

ガイド

  • PyCharm ヘルプ -> ユーザー・パラメーター

  • PyCharm Help -> User Parameters

手順

  1. Database Console 上で スパナのボタン ( Settings ) -> User Parameters

    (もしくは、 Preferences -> Tools -> Database -> User Parameters)

  2. こう

    • In scripts を ON にしないと、 Database Console で使用できない

  3. Apply ボタンを押下

  4. OK ボタンを押下
